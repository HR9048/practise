Overflow

defination: the overflow property is used to control how content overflows the boundaries of its containing element when the content is larger than the container's dimensions. It is commonly used with elements that have a fixed size or limited space to handle content that cannot be fully displayed within that space.

The overflow property can take the following values:

1 visible: This is the default value. Content overflows the container and is visible outside the box. It may overlap other elements.

example: .container {
width: 200px;
height: 100px;
overflow: visible;
}

2 hidden: Content that overflows the container is clipped and not visible outside the box. This effectively hides any content that exceeds the container's dimensions.

example: .container {
width: 200px;
height: 100px;
overflow: hidden;
}

3 scroll: A scrollbar is added to the container, whether it's needed or not. Users can scroll to see the overflowing content.

example: .container {
width: 200px;
height: 100px;
overflow: scroll;
}

4 auto: A scrollbar is added to the container only if the content overflows. If the content fits within the container, no scrollbar is displayed.

example: .container {
width: 200px;
height: 100px;
overflow: auto;
}
