Pseudo-element selector

defination: A CSS pseudo-element is a keyword added to a selector that lets you style a specific part of the selected elements. For Example, Styling the first letter or line of an element, and Inserting content before or after the content of an element. All of these can be done using Pseudo Elements in CSS.

::first-line element

This is a paragraph using first-line pseudo-element to style first line of the paragraph. Content in the first line turns red and becomes bold.
