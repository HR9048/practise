calci in css

defination: "calc" is a mathematical function that allows you to perform simple arithmetic calculations to determine the value of a CSS property. The calc function is useful when you need to dynamically calculate a value based on other property values or fixed values.

syntex: calc(expression)

example: /* The element's width will be 100% of the parent's width minus 200 pixels */

width: calc(100% - 200px);
