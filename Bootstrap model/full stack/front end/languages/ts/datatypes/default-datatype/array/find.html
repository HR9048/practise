Find Method

defination: This method is used to find the first element in an array that satisfies a given condition. It returns the first matching element, or undefined if no matching element is found.

Find Method

const foundElement = array.find(element => condition);


Example for Find Method


          const books = [
            { title: "Dune", genre: "Science Fiction" },
            { title: "1984", genre: "Dystopian" },
            { title: "Foundation", genre: "Science Fiction" },
            { title: "Brave New World", genre: "Dystopian" }
          ];
          
          // Using the find method to find the first Science Fiction book
          const scienceFictionBook = books.find(book => book.genre === "Science Fiction");
          
          if (scienceFictionBook) {
            console.log("Science Fiction book found:", scienceFictionBook.title);
          } else {
            console.log("No Science Fiction book found.");
          }
