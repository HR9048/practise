External CSS

defination: External Styles (Linked Stylesheets): External styles are defined in separate CSS files with a .css extension. These CSS files are then linked to the HTML document using the 'link' element in the 'head' section. External stylesheets offer the best separation of concerns, as they allow you to keep the styles separate from the HTML structure, promoting maintainability and reusability.
