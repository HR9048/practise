Responsive Design

Defination: Responsive design in CSS refers to a web development approach that aims to create websites or web applications that adapt and respond to various screen sizes and devices. It ensures that the layout and content of a web page adjust automatically based on the user's screen resolution, making the site look aesthetically pleasing and easy to use on desktops, tablets, smartphones, and other devices.

Types in Responsive design

  1. Media Query
  2. Flex design
  3. grid design
