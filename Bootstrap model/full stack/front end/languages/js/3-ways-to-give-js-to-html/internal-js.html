Internal Script

defination: An internal script refers to JavaScript code that is directly written within the 'script' tags within the HTML document itself. This is also sometimes called "inline script.

Internal Script

<!DOCTYPE html>
<html>
<head>
    <title>Internal Script Syntax</title>
    <script>
        // Your JavaScript code here
    </script>
</head>
<body>
    <!-- HTML content -->
</body>
</html>


Example of inline Script

    <!DOCTYPE html>
    <html>
    <head>
        <title>Internal Script Example</title>
    </head>
    <body>
        <h1 id="message">Hello, World!</h1>
        
        <button onclick="changeMessage()">Change Message</button>
        
        <script>
            // Inline JavaScript code
            function changeMessage() {
                var messageElement = document.getElementById("message");
                messageElement.innerHTML = "Hello from Inline Script!";
            }
        </script>
    </body>
    </html>
