group selector

defination: A group selector, also known as a multiple selector, allows you to apply the same CSS styles to multiple selectors in a single rule. It is a convenient way to write concise and efficient CSS code when you want to apply identical styles to various elements. Grouping selectors can help reduce redundancy and improve the maintainability of your CSS code.

types of group selectors

  1. Universal selector
  2. Attribute selector
  3. Pseudo-class selector
  4. Pseudo-element selector
