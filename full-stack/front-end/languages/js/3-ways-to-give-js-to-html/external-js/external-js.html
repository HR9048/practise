External Script

defination: The other way is to write JavaScript code in another file having a .js extension and then link the file inside the <head> or <body> tag of the HTML file in which we want to add this code.

External Script

<!DOCTYPE html>
<html>
<head>
    <title>External Script Example</title>
    <script src="path/to/your/script.js"></script>
</head>
<body>
    <!-- HTML content -->
</body>
</html>


Example for External Script

/* styles.css */
body {
background-color: lightgray;
font-family: Arial, sans-serif;
}

h1 {
color: blue;
}

p {
color: darkgreen;
}
    
<!DOCTYPE html>
<html>
<head>
    <title>External CSS Example</title>
    <link rel="stylesheet" type="text/css" href="path/to/your/styles.css">
    <script src="path/to/your/script.js"></script>
</head>
<body>
    <h1>This is a heading</h1>
    <p>This is a paragraph.</p>
</body>
</html>
