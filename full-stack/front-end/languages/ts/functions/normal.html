Normal Function

defination: A normal function in TypeScript is a function that is defined with the function keyword and can be called by its name.

Normal Function

function functionName(parameter1: Type1, parameter2: Type2): ReturnType {
  // Function body
}

Example for Normal Function

          
function add(a: number, b: number): number {
  return a + b;
}
const result = add(5, 3);
console.log(result); // Output: 8
