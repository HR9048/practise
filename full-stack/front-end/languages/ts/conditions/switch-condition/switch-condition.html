Switch Statement

defination: A switch statement is a control flow statement used to evaluate an expression against multiple possible case values. It provides a way to execute different code blocks based on the value of the expression, similar to a series of if-else if-else statements, but in a more concise and structured manner.

Switch Statement

switch_expression = value
if switch_expression == case1:
    # Code for case1
elif switch_expression == case2:
    # Code for case2
# Add more elif blocks for additional cases
else:
    # Code for the default case



Example for Switch Statement

          
day = "Monday"
if day == "Monday":
    print("It's the start of the workweek.")
elif day == "Friday":
    print("It's almost the weekend.")
else:
    print("It's a regular day.")
