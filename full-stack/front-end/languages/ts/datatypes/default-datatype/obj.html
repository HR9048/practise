Obj Data Type

defination: In both TypeScript and JavaScript, an "object" is a fundamental data structure that allows you to group related data and functionality together. Objects consist of key-value pairs, where the keys are strings (or symbols in some cases) that act as property names, and the values can be of any data type, including other objects, functions, or primitive values.

Obj

Syntx in TypeScript
let objectName: { key1: type1, key2: type2 } = {
key1: value1,
key2: value2,
// More key-value pairs
};
Syntax in JavaScript
let objectName = {
key1: value1,
key2: value2,
// More key-value pairs
};

Example for Obj


Example of Obj in TypeScript

                    
let person: {
    firstName: string;
    lastName: string;
    age: number;
    sayHello: () => void;
} = {
    firstName: "John",
    lastName: "Doe",
    age: 30,
    sayHello: function() {
        console.log("Hello!");
    },
};
