Number Data Type

defination: In JavaScript and TypeScript, the number data type is used to represent numeric values, including both integers and floating-point numbers (numbers with decimal points). It is one of the fundamental data types for working with numerical data.

Number

Syntx in TypeScript
let variableName:Number = numericValue;
Syntax in JavaScript
let variableName = numericValue;

Example for Number


Example of number in TypeScript

                    
let integerNumber: number = 42;        // Integer
let floatingPointNumber: number = 3.14; // Floating-point
let negativeNumber: number = -10;       // Negative number

Example of number in JavaScript

                    
let integerNumber = 42;        // Integer
let floatingPointNumber = 3.14; // Floating-point
let negativeNumber = -10;       // Negative number
