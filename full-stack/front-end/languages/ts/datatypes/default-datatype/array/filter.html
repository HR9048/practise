Filter Method

defination: This method creates a new array containing all elements that satisfy a given condition. It returns an array with elements that passed the condition.

Filter Method

filtered_iterator = filter(condition_function, iterable)


Example for Filter Method


           // Sample array of numbers
            const numbers = [5, 10, 15, 20, 25];
    
            // Function to filter out even numbers
            function filterEvenNumbers() {
                const evenNumbers = numbers.filter(number => number % 2 === 0);
                if (evenNumbers.length > 0) {
                    document.getElementById("result").textContent = "Even numbers: " + evenNumbers.join(", ");
                } else {
                    document.getElementById("result").textContent = "No even numbers found.";
                }
            }
