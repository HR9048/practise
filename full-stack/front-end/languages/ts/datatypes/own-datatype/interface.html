Interface

defination: An interface in TypeScript defines a contract or blueprint for an object's structure. It specifies which properties and methods an object should have. Interfaces are particularly useful for defining the shape of objects that should adhere to a specific structure.


Example for Interface

            
interface Person {
    firstName: string;
    lastName: string;
    age: number;
}
                  
const john: Person = {
    firstName: "John",
    lastName: "Doe",
    age: 30,
    };
